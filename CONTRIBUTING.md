# Contributing to Hookflow

Thanks for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR-USERNAME/hookflow.git`
3. Install dependencies: `npm install`
4. Copy `.env.example` to `.env.local`
5. Start the dev server: `npm run dev`

## Making Changes

1. Create a branch: `git checkout -b feature/your-feature`
2. Make your changes
3. Run the build: `npm run build`
4. Commit with a clear message
5. Push and open a Pull Request

## Ideas for Contributions

- Add real Supabase integration for persistent storage
- Add real email sending via Resend
- Add more trigger types (GitHub events, Slack messages)
- Add more action types (Slack webhooks, Discord)
- Add a visual flow editor with reactflow
- Write tests
- Improve error handling
